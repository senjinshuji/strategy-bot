# 広告生成コマンド

戦略分析結果から具体的な広告クリエイティブを生成

## バナー広告生成
```bash
# 静的バナー案生成
generate-banner [サイズ] [訴求ポイント]

# 動的バナー案生成
generate-dynamic-banner [ターゲット]
```

## 動画広告生成
```bash
# 15秒動画構成案
generate-video-15s

# 30秒動画構成案  
generate-video-30s
```

## LP改善案生成
```bash
# ファーストビュー改善
optimize-first-view

# 全体構成最適化
optimize-lp-structure
```
